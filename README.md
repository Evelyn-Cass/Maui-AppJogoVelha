# 🎮 Jogo da Velha - MAUI App

A modern, cross-platform Tic-Tac-Toe game built with .NET MAUI. Play against a friend on Android, iOS, Windows, or macOS with a clean and intuitive interface.

## ✨ Features

- 🎯 **Two-player gameplay** - Pass and play with a friend
- 🎨 **Modern UI design** - Clean, responsive interface with animations
- 📱 **Cross-platform** - Works on Android, iOS, Windows, and macOS
- 🏆 **Win detection** - Automatically detects rows, columns, and diagonals

## 🛠️ Technologies

- **.NET 10.0** - Latest .NET framework
- **.NET MAUI** - Multi-platform App UI
- **C#** - Primary programming language
- **XAML** - UI markup language
